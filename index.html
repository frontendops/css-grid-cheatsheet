CSS Grid Demo

Standard 12 Column Grid

                            
.item-1 {
    grid-column-start: 1;
    grid-column-end: 8;
}
                            
                        
                            
.container {
    column-gap: 1rem;
    display: grid;
    grid-template-columns: repeat(12, 1fr);
}

.item-2 {
    grid-column-start: 8;
    grid-column-end: 13;
}

Two Dimensional Grid

                            
.item-1 {
    grid-column: 1 / span 8;
}
                            
                        
                            
.container {
    grid-template-columns: repeat(12, 1fr);
    grid-template-rows: 1fr 1fr;
    row-gap: 1rem;
}

.item-2 {
    grid-column: 9 / span 12;
    grid-row: 1 / span 2;
}
                            
                        
                            

.item-3 {
    grid-column: 1 / span 8;
}

Resonsive Layout Grid (no media-queries)

                            
.container {
  grid-template-columns: 
  repeat(auto-fit, minmax(300px, 1fr));

  grid-template-rows: 
  repeat(auto-fit, minmax(174px, 1fr));
}
